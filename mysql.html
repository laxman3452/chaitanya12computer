MySQL Syllabus Reference Guide

Quick formulas, syntax structures, and execution blueprints for examinations.

1. Structure Commands (DDL)

DDL : CREATE

Create Table Syntax

Defines column constraints and schema mappings.

CREATE TABLE student (
    roll INT,
    first_name VARCHAR(20),
    last_name VARCHAR(20),
    age INT
);
DDL : ALTER

Alter Table Operations

Modifies schema structural design properties dynamically.

/* Add column */
ALTER TABLE student ADD DateOfBirth DATE;

/* Drop column */
ALTER TABLE student DROP COLUMN age;

/* Modify column type */
ALTER TABLE student MODIFY COLUMN DateOfBirth YEAR;

2. Data Manipulation Commands (DML)

DML : INSERT

Insert Statement

Appends value sets into defined database collection arrays.

INSERT INTO student (roll, first_name, last_name, DateOfBirth) 
VALUES (1, 'Aayushi', 'Karn', 2010);
DML : SELECT

Select Queries & Sorting

Filters data streams using logical evaluation assertions.

/* Filtered query matching structural constraints */
SELECT roll, first_name FROM student 
WHERE roll = 20;

/* Output sorting execution structural rule */
SELECT * FROM student 
ORDER BY roll ASC;
DML : UPDATE / DELETE

Data Modification & Purging

Warning: Always target accurate references with a WHERE clause handler.

/* Update entry */
UPDATE student 
SET first_name = 'Alok', last_name = 'Kumar' 
WHERE roll = 1;

/* Delete row sequence mapping */
DELETE FROM student 
WHERE roll = 20;
