Layer-by-Layer Breakdown

Layer 7: Application Layer Data Unit: Data
What it does: It acts as the window for applications to access network services. It interacts directly with the software you use.
Protocols/Examples: HTTP (Web browsers), SMTP (Email), FTP (File transfer).
Layer 6: Presentation Layer Data Unit: Data
What it does: It translates, encrypts, and compresses data. It makes sure the receiving device can read the data structure.
Key Functions: Data Encryption (security) and Data Compression (reducing file size).
Layer 5: Session Layer Data Unit: Data
What it does: It sets up, manages, and terminates connections (sessions) between two devices. It acts as a coordinator to keep the conversation going smoothly.
Layer 4: Transport Layer Data Unit: Segments
What it does: It ensures reliable, end-to-end data delivery. It breaks large data into smaller pieces called segments and handles error checking.
Key Functions: Flow control (preventing a fast sender from overwhelming a slow receiver) and error correction.
Protocols: TCP (Reliable) and UDP (Fast but unreliable).
Layer 3: Network Layer Data Unit: Packets
What it does: It is responsible for routing data from the source host to the destination host across different networks. It handles logical addressing.
Key Functions: Routing (finding the best path) and IP Addressing.
Devices/Protocols: Routers operate here; IP protocol.
Layer 2: Data Link Layer Data Unit: Frames
What it does: It ensures error-free data transmission from one node to the next over the same physical link. It packages network packets into frames.
Key Functions: Physical addressing (MAC Address) and error detection using a checksum.
Devices: Switches and Network Interface Cards (NIC) operate here.
Layer 1: Physical Layer Data Unit: Bits (0s and 1s)
What it does: The lowest layer, dealing with the actual physical medium (hardware) that carries the electrical, optical, or radio signals.
Key Functions: Defining cable types, voltages, pin layouts, and data transmission speeds.
Devices: Hubs, repeaters, ethernet cables, and fiber optics.
