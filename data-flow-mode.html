Exam Notes: Communication Modes

1. Introduction to Communication Mode

  • Definition: Communication mode (also known as transmission mode) defines the direction of data flow between two connected devices in a network.
  • Purpose: It determines how data is directed, routed, and managed across a communication channel.
  • Classification: There are three primary types: Simplex, Half-Duplex, and Full-Duplex.

2. Types of Communication Modes

1. Simplex Mode

  • Direction of Flow: Completely unidirectional (one-way communication).
  • Mechanism: One device can only send data, while the other device can only receive data. The receiving device cannot send any data back.
  • Channel Capacity: The entire capacity of the communication channel is dedicated to transmitting data in that single direction.
  • Analogy: Similar to a one-way street where traffic can only move in one legal direction.
  • Simple Example: A Keyboard and CPU. The keyboard can only type and send data to the CPU, but it can never receive text inputs back from the CPU.

2. Half-Duplex Mode

  • Direction of Flow: Bidirectional (two-way communication), but not simultaneously.
  • Mechanism: Both devices can send and receive data, but only one device can transmit at any given instant. While one device sends, the other must listen.
  • Channel Capacity: The full capacity of the transmission medium is completely utilized by whichever device is currently sending.
  • Analogy: Similar to a narrow one-lane bridge. Traffic can go both ways, but cars from one direction must stop and wait for oncoming cars to pass first.
  • Simple Example: A Walkie-Talkie. One person presses a button to speak while the other listens. If both try to talk at the exact same time, the messages collide and cannot be heard.

3. Full-Duplex (Duplex) Mode

  • Direction of Flow: Bidirectional (two-way communication) simultaneously.
  • Mechanism: Both connected devices can send and receive data at the exact same time without needing to wait for each other.
  • Channel Capacity: The total capacity of the communication line is shared/divided into two distinct paths to accommodate concurrent incoming and outgoing traffic.
  • Analogy: Similar to a multi-lane two-way highway where vehicles travel in opposite directions at the same time smoothly.
  • Simple Example: A Mobile Phone Call. Two people talking over smartphones can speak and hear each other at the exact same instant without waiting.
