Unit 2: Data Communication and Networking

1. Concept of Communication System

Definition

The term communication refers to the sending, receiving, and processing of information by electronic media.

Historical Evolution

  • 1840s: Started with wire telegraphy.
  • Later Decades: Developed into telephony (telephone systems).
  • Beginning of 20th Century: Radio communication was introduced, made possible by the invention of the triode tube (vacuum tube).
  • World War II: Stimulated rapid growth, leading to wider use and refinement.
  • Mid-20th Century: Enhanced by the invention of transistors, integrated circuits (ICs), and other semiconductor devices.
  • Modern Era: Shifted toward the use of satellites, fiber optics, computers, and advanced digital data communications.
Example:

Making a cellular phone call uses a modern communication system where your voice is processed into a digital signal, transmitted wirelessly via radio waves to a tower, routed through fiber-optic cables, and received by another phone.

2. Computer Network

Definition

A computer network is defined as two or more computing devices linked together to communicate and share information (such as files and software programs) and hardware resources (such as printers, hard disks, and plotters).

Key Terms

  • Networking: The concept of connected computers sharing resources.
  • Node: Any computer or device connected to a network that can communicate with other users or devices.
  • Mediums of Connection: Can be wired (cables) or wireless (Bluetooth, radio links, Wi-Fi, satellite, Infrared ports).
Example:

An office room containing five computers connected together via a Wi-Fi router to share a single central printer forms a local computer network.

3. Advantages of Using Computer Network

  1. Resource Sharing: Sharing hardware and software resources among multiple users. This reduces costs because every individual standalone computer does not need its own dedicated peripherals.
    Example:

    Instead of buying ten separate printers for ten employees, a company installs one high-end laser printer connected to the network for everyone to use.

  2. Centralized Control and Management: All computers in a network are typically connected to a central server. This server defines policies, updates software, and enforces security measures globally for all network members.
    Example:

    A network administrator can block unauthorized websites on all office computers simultaneously by changing settings on the main server.

  3. Speedy and Cost-Effective Communication: Users can send messages, files, and documents directly from one computer to another within seconds, eliminating manual delivery costs.
    Example:

    An employee sending an instant text message or email to a colleague in another room over the office network instead of printing out a memo.

  4. Backup and Recovery: Data can be securely and systematically stored on a central network server. If an individual node experiences a system crash, the data remains safe and can be easily recovered from the server.
    Example:

    If an employee's laptop breaks down completely, they can log into another computer on the network and instantly access their saved files from the server backup.

  5. Flexible Access: Users are not tied to one single machine; they can securely access their personal work files and applications from any authorized terminal connected to the network.
    Example:

    A student logging into their school account from a computer in the library and accessing the same files they saved while working in the computer lab.

  6. Workgroup Computing: Specialized workgroup software (such as collaborative office suites) allows multiple users in different locations to collaborate, edit, and contribute to the same document simultaneously.
    Example:

    Three teachers from different schools working together in real-time online to draft a new educational curriculum document.

4. Disadvantages of Using Computer Network

  1. Expensive setup: Connecting computers requires additional specialized hardware infrastructure such as Network Interface Cards (NICs), hubs, switches, routers, and cabling, which increases initial installation and operational costs.
    Example:

    Setting up a new cyber cafe requires a high initial budget to purchase routers, switches, LAN cables, and hire technicians to install them.

  2. Security Problems: Because data is transmitted over a shared public or private medium, it is vulnerable to unauthorized access. Interconnected systems experience an increased risk of computer crimes such as viruses, hacking, data theft, software piracy, plagiarism, and spoofing.
    Example:

    A single computer on a local network downloads a malicious virus from an email attachment, which then rapidly spreads and infects all other computers on the same network.

  3. Needs Special Technical Knowledge: Designing, installing, managing, and maintaining a secure, functional network system requires trained, skilled manpower (Network Administrators).
    Example:

    When a company's network security settings need an update, they must hire a certified network engineer because standard office staff do not know how to configure routers.

  4. Network Failure: If the central network hub, switch, or main server fails, the entire communication system can stall, halting day-to-day business operations.
    Example:

    If the central server of a commercial bank crashes, tellers cannot process cash deposits or withdrawals, bringing bank transactions to a complete stop.

  5. Complex: A computer network is inherently complicated to configure, establish, troubleshoot, and operate compared to isolated standalone computers.
    Example:

    When one computer loses internet access, finding the root cause requires checking multiple points: the network cable, the wall jack, the local switch, the router settings, or IP address conflicts.
